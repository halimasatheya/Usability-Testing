Usability Testing Tools

Usability testing is a technique used in user-centered interaction design to evaluate a product by testing it on users.

This website will show the different usability testing tools. Using the navigation below explore each page and how the usability testing tool relates to the process of creating a cake delivery app.

1) Usability Testing

Usability testing involves testing a product an whether it can be used appropriately for it's intended purpose under controlled conditions to determine how well users can use a product.

Read More

2) Focus Groups

Focus Groups is a discussion that takes place with intended users to gain insight on their views on a product or system. Users are asked for their opinions and ideas.

Read More

3) Card Sorting

Card Sorting is a quick method which is used to help design information architecture for a website or application. There are two types of Card Sorts either open card sort or closed card sort.

Read More

4) Prototyping

Prototyping a method used to test designs in which a few elements of the interface are working. Used as part of usability testing to gain users feedback on the design. A prototype can be a low or high fidelty prototype.

Read More

5) Questionnaires

A set of questions in which user feedback is gained regarding a product or service which has been tested to gain the users opinions. A cheap quick method which provides both quantitative and qualitative data.

Read More

6) Think Aloud

A method used in which users thoughts and ideas are gathered in a usability test. User is asked to talk aloud when completing tasks, describing thier thoughts they are thinking in the current moment.

Read More

7) Heuristic Evaluation

A usability testing tool which allows you to identify usability problems in user interface designs. Uses well known heuristic principles developed by Jakob Nielsen to evaluate a system. A beneficial method to use.

Read More

8) Eye Tracking

Eye tracking is the process of measuring either the point of gaze (where one is looking) or the motion of an eye relative to the head. An eye tracker is a device for measuring eye positions and eye movement.
