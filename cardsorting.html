Card Sorting

What is Card Sorting?


Card sorting is a usability tool that is used in user experience design, usually used when designing information structure, navigation website,
menu structure. It is used when a variety of content needs to be sorted i.e. structured into correct categories.
The card sort allows us to gain different users perception on how they personally would navigate through the site.
This can be crucial information for developers who want to know how intended users would personally like the website or menu structure to be.

Why you should use card sorting?

The main aim of card sorting is to know whether the current information structure or proposed navigation you have planned for a website or app is correct and whether its working for your users. Because if your customers are having a bad user experience due to navigation not being structured properly your loosing customers and gaining a bad reputation at the same time. The card sorting tool allows you to make sure that the user is able to find the product or the goal for which they came on to your website to achieve. By never conducting a card sort you'll never know what your users think about the information structure and categories for a website navigation.

Open Card Sort

An open card sort is when the intended user is given all index cards and the user is asked to arrange the cards in a suitable way in which they think relates. So the user groups the cards and the user is then asked to names the group of cards in a name in which they think is most appropriate for each category.

Closed Card Sort

A closed card sort is when the intended user is given both index cards and categories and is asked to sort the index cards into the categories, which they think best fit. Closed card sorting allows you to see whether the categories you already have in place are correct also which index cards the user thinks best fits in each category.

Benefits of Card Sorting

  • Allow you to know whether the index cards in a current category are correct according to the users opinion
  • Allow you to gain insight in the intended users opinion of how the structure should be structured
  • Gain feedback from the user on certain cards they think didn't fit in which group
  • Provides you with valuable information as a business it provides you with users opinions on how they personally would like the hierarchy of information structure to be presented
  • It isn't time constraining
  • Costs to conduct card sorting session is not expensive at all
  • Anything confusing or difficult for end user can be identified
