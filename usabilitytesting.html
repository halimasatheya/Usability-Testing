What is Usability Testing?

Usability testing involves testing a product an whether it can be used appropriately for it's intended purpose.

So before we start on our journey on creating this usable cake delivery app its important you understand what usability testing is!
Usability Testing involves testing a product an whether it can be used appropriately for it's intended purpose. Different methods can be used either to improve a existing product or for a new proposed product.

Benefits of Conducting Usability Testing

  • Allows you to identify and address problems in early stages of development
  • Critical problems are apparent immediately when conducting usability testing
  • Identify any functionality that is missing within a product/service
  • Can detect errors much earlier in the development life cycle
  • Gain quick user feedback which means better user experience for users
  • Able to identify any changes that need to be considered to improve user satisfaction and perfomance
  • Saves time and extra costs from having to recreate a finished product
